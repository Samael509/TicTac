## ЕСЛИ ПРОГРАММА ДОЛГО ЗАПУСКАЕТСЯ
программа может запускаться долго (у меня 30 сек.) из-за того что я использовал алгоритм min-max для сложного бота
Отключение сложного бота:
1. открыть MainWindow.xaml.cs (любой редактор)
2. найти метод BotMove() (189 строка)
3. внутри "switch" для "Difficulty.Hard" заменить moveIndex = HardBot(); на moveIndex = EasyBot();/MediumBot(); или MediumBot() ?? EasyBot();
4. готово
